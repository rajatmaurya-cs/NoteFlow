<h2 align="center">🌟 Uses of NoteCraft</h2>

<p align="center">
<b>AI Powered NoteCraft</b> is specially designed for students to improve productivity and learning using Artificial Intelligence.
</p>

<p align="center">
📝 Create and organize smart notes <br>
🤖 Multiple AI Models available
🤖 Solve academic questions instantly using AI <br>
📥 Save AI answers directly into notes <br>
📅 Manage daily tasks and schedules <br>
📈 Track study progress with charts <br>
🌙 Enjoy smooth premium dark UI <br>
⚡ Increase focus and productivity
</p>
